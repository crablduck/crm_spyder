## 🌐 五、数据源清单

### 高优先级（必须）

```
1. 福建省政府采购网 (ccgp-fujian.gov.cn)
   → 最权威，信息最完整
   
2. 中国政府采购网-福建站 (ccgp.gov.cn)
   → 补充数据
   
3. 医院官网新闻
   → 系统上线新闻、信息化建设报道
```

### 中优先级（重要）

```
4. 各地市政府采购网
   → 福州、厦门、泉州等地市级
   
5. 医院公众号
   → 宣传报道
   
6. 供应商公众号
   → 案例分享
```

### 低优先级（可选）

```
7. HC3i数字医疗网
8. CHIMA案例库
9. 行业展会新闻
```
